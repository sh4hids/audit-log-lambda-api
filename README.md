# audit-log-lambda-api

### Clone and prepare kit

```bash
git clone git@github.com:sh4hids/audit-log-lambda-api.git
```

```bash
cd audit-log-lambda-api
```

```bash
rm -rf .git
```

```bash
git init
```

```bash
cp .env.dev .env
```

### Install dependencies

```bash
npm i
```

### Add git hook

```bash
npm run prepare
```

### Start dev server

```bash
npm run dev
```

### Build project

```bash
npm run build
```

### Start prod server

```bash
npm start
```
